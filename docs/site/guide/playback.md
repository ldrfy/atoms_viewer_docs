# 多帧播放与录制

## 多帧数据

XYZ 与 LAMMPS-dump 均可包含多帧，用于轨迹回放。帧数很大时建议分段加载或降低播放 FPS。

## 播放控制

播放条支持播放/暂停、拖动定位与输入帧号跳转。切换图层可见性不影响播放控制。

## 帧重建

可开启“逐帧刷新键线”以反映键连接变化；关闭可显著提升播放性能。

## 视频录制

点击录制后框选录制区域，可拖拽调整。录制帧率在“其他设置”中配置，输出为 WebM。
需要更小体积时，可降低帧率、缩小区域或缩短时长。

## 播放建议

- 在关键帧处暂停并进行测量或导出。
- 隐藏无关图层以提升播放流畅度。
