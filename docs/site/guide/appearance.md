# 图层外观与配色

## 图层独立与当前图层提示

颜色、键线、原子大小等参数按图层独立保存。
“原子颜色”和“图层外观”面板会提示当前图层，避免误改。
可开启“同时编辑所有图层”以同步统一风格。

## 原子颜色映射

支持元素级或元素+typeid 组合映射（如 C1、O2）。
颜色可通过十六进制输入或色盘选择，并可恢复默认。
开启同步后可一键统一多图层配色。

## 图层外观参数

- 原子大小：整体缩放球半径。
- 键显示：关闭可显著提升性能。
- 键长阈值（bondFactor）：控制自动成键范围。
- 球体细分：平滑度与性能的权衡。
- 键线半径：增强可读性。

建议：探索时关闭键线并降低细分，出图阶段再提高质量。

## 一致性建议

多图层对比时，先确认各图层 LAMMPS 映射无误，再统一配色与外观。

## 性能相关建议

- 大模型可关闭键线以显著提升性能。
- 交互阶段降低球体细分，导出阶段再提高。
- bondFactor 过大会产生过多键线，影响可读性。
