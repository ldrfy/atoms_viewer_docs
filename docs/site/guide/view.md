# 视图与交互

## 单视图与双视图

单视图用于专注观察，双视图用于同时对照两个投影方向，便于理解厚度方向、晶体取向与缺陷形态。

## 双视图设置

在“显示控制”中勾选两个视角即可进入双视图，可自由组合正视/侧视/俯视，并调整分屏比例。

## 联动交互

双视图支持同步旋转、缩放与选中，保证两个视角一致变化。
如需独立视角调整，建议切回单视图。

## 鼠标与触控

- 滚轮：缩放。
- 左键拖拽：旋转。
- 右键/中键拖拽：平移。
- 触控：单指旋转、双指缩放。

## 精确视角控制

可输入 XYZ 旋转角与视距获得可复现视角。
“恢复原始视距”用于回到初始 fit 视距但保留当前旋转。
可切换透视/正交投影以满足不同展示需求。

## 预设视角与重置

- 预设视角可快速切换到正视/侧视/俯视。
- “回到旋转前”用于恢复初始角度。
- “恢复原始视距”仅重置视距不改变角度。

## 视角复现建议

- 先交互探索，再记录 XYZ 角度与视距。
- 多图层对比先统一视角，再微调外观与映射。
- 导出前确认背景色与投影方式一致。
