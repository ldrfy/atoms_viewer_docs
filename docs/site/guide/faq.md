# 常见问题与排查

## 加载卡住或动画不消失

- 大文件解析需要时间，请耐心等待。
- 查看控制台是否有异常报错。
- 尝试逐个加载以定位问题文件。
- 更换浏览器排除兼容性差异。

## 旋转或缩放不流畅

- 可见图层过多或播放 FPS 过高会导致卡顿。
- 隐藏图层、降低复杂度或使用单视图。

## LAMMPS 映射不符合预期

- 确认当前编辑的是目标图层。
- 修改后点击“刷新显示”。
- 检查文件中的 typeid 与映射表一致。

## 导出很慢或失败

- 降低导出倍率或隐藏不必要图层。
- 只在需要时使用透明背景。
- 双视图导出开销更高，可切回单视图。

## 透明 PNG 仍有空白

- 确认透明背景已开启。
- 远处离散原子可能扩大裁切范围，尝试隐藏无关图层。

## 录制视频无法下载或播放

- 确认浏览器下载未被阻止。
- WebM 可转码为 MP4。
- 降低帧率或缩小录制区域后重试。
