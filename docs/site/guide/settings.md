# 个性化与系统设置

## 多语言与自动切换

界面支持多语言并可根据系统语言自动切换，也可在顶部栏手动选择。
语言切换不会影响数据与设置。

## 深色与浅色主题

深色主题适合长时间观察，浅色主题适合论文截图。
当背景与主题对比不足时可提示可读性问题。

## 其他设置

“其他设置”包含坐标轴显示、录制帧率等通用选项。
大型模型建议降低球体细分或关闭键线以提升性能。

## 同步与一致性

- 需要统一风格时可开启“同时编辑所有图层”。
- 若出现可读性提示，可调整背景色或主题。

## 本地设置保存与清理

常用设置会保存在本地浏览器中，可通过“清理本地设置”恢复默认。
支持导出/导入设置 JSON 以便跨设备迁移。

## 本地计算与隐私

所有解析与渲染均在本地完成，分享时建议导出图片或视频。
