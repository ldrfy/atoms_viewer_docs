# 功能总览

## 你可以用它做什么

- 加载 XYZ/PDB/LAMMPS 等结构与轨迹文件进行交互式查看。
- 多文件作为图层叠加显示以便对比。
- 双视图联动对照不同投影方向。
- 原子拾取查看类型、编号与坐标信息。
- 计算距离与夹角等几何量。
- 播放多帧轨迹并导出关键帧。
- 导出 PNG（透明/纯色背景）与录制视频。
- 语言与主题切换。

## 10 分钟快速上手

1. 打开网页或本地版本，点击“打开文件”或拖拽文件进入页面。
2. 在“图层面板”确认已加载文件，并切换当前编辑图层。
3. 对 LAMMPS 数据完成 typeid → 元素映射并点击“刷新显示”。
4. 在“原子颜色/图层外观”调整配色、原子大小、键显示等参数。
5. 鼠标旋转/缩放/平移，或输入精确 XYZ 角度与视距。
6. 使用播放条控制轨迹，并导出 PNG 或录制视频。

## 界面组成与术语

- 主画布：模型渲染与交互。
- 右侧面板：文件、图层、显示、播放、导出等设置。
- 状态提示：解析、重建与导出进度。
- 播放条：多帧控制与录制。

术语约定：

- 当前图层：面板编辑目标图层。
- 活动图层：拾取与测量的对象图层。
- 可见图层：当前渲染显示的图层集合。

## 推荐工作流

1. 加载文件并确认解析结果（格式、原子数、帧数）。
2. 对 LAMMPS 数据完成映射并重建图层。
3. 调整视角与投影方式，固定可复现视角。
4. 统一配色与外观参数，锁定风格。
5. 在活动图层上进行拾取与测量。
6. 播放轨迹定位关键帧并导出 PNG 或录制视频。

## 输出场景

- 论文插图：透明背景 PNG + 自动裁切。
- 汇报展示：纯色背景 PNG 或录制短视频。
