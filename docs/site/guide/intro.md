# 前言

本手册面向使用兰朵儿原子查看器（LDR Atoms Viewer）进行原子结构可视化与轻量分析的科研用户。
内容以“快速上手、操作可复现、便于论文出图”为组织原则，并以 v1.0.0 为准。
若界面或文案与示例略有差异，请以实际界面为准并理解操作意图。

## 软件定位与适用范围

本工具以浏览器端渲染为核心，适用于材料模拟、表界面结构、缺陷演化、纳米体系
以及一般原子级三维可视化场景。典型输出为论文插图 PNG、汇报展示视频等。

## 运行方式与数据安全

解析与渲染完全在本地完成，模型文件不会上传到服务器。需要共享时建议导出图片或视频。

## 软硬件环境建议

- 支持 WebGL 的现代浏览器（Firefox/Chrome/Edge）。
- 多核 CPU 与充足内存以应对大型轨迹。
- 独立显卡优先，百万级原子更流畅。

建议配置：

- 浏览器：Firefox/Chrome/Edge 最新版，WebGL 与视频编码更稳定。
- 处理器：4 核及以上，解析与重建更高效。
- 内存：16GB 及以上，多图层与长轨迹更稳。
- 显卡：独立显卡优先，渲染更流畅。

## 开始前的小建议

- 大文件解析可能需要几十秒，请等待状态提示完成。
- 显示异常时先检查 LAMMPS 映射并点击“刷新显示”。
- 导出前隐藏无关图层并锁定视角。
