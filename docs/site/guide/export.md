# 导出与分享

## PNG 导出与背景

PNG 可用于论文与汇报。支持纯色背景或透明背景，透明时会自动裁切空白。

## 分辨率与缩放倍率

提高导出倍率可获得更细腻边缘，但会增加耗时与显存占用。
建议预览阶段使用默认倍率，定稿阶段再提升倍率。

## 导出前检查清单

- 视角与视距是否锁定。
- 只保留需要的可见图层。
- 背景色/透明设置是否符合排版要求。
- 多图一致时确认配色与外观统一。

## 双视图导出

双视图导出会包含两个视图的组合图，可调整分屏比例。
只需单视图时请切回单视图后导出。

## 视频导出建议

选择稳定背景、适度分辨率与合理帧率，避免过长无变化片段。

## 出图质量建议

- 论文插图优先使用透明背景。
- 多张图保持相同视角与外观参数。
- 仅保留必要图层，避免裁切范围过大。
